1. O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação de documentos HTML.

Como funciona o CSS?

Seleciona elementos HTML e aplica estilos a eles.

Sintaxe básica do CSS

selector { 
    property: value; 
  }

2. Prioridades no CSS

No CSS, a prioridade determina quais estilos serão aplicados a um elemento quando há múltiplas regras que poderiam se aplicar. A prioridade é influenciada por três fatores principais:

  1. Especificidade
  2. Importância
  3. Ordem de declaração

1. Especificidade

A especificidade é calculada com base no tipo de seletor utilizado:

  • Seletores de elemento (tag): têm menor especificidade.
  • Seletores de classe: têm especificidade média.
  • Seletores de ID: têm maior especificidade.
  • Estilo inline: aplicado diretamente no elemento HTML, tem a maior especificidade.

Exemplo:

<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    /* Seletores de elemento */
    p {
      color: blue; /* Especificidade baixa */
    }

    /* Seletores de classe */
    .important-text {
      color: green; /* Especificidade média */
    }

    /* Seletores de ID */
    #unique-text {
      color: red; /* Especificidade alta */
    }
  </style>
</head>
<body>
  <p id="unique-text" class="important-text">Texto de exemplo</p>
</body>
</html>

Resultado: O texto será vermelho porque o seletor de ID tem maior especificidade.

2. Importância

A propriedade !important pode ser usada para forçar um estilo a ter prioridade máxima, independentemente da especificidade dos seletores.

Exemplo:

p {
  color: blue !important;
}

#unique-text {
  color: red;
}

Resultado: O texto será azul porque !important tem prioridade máxima.

3. Ordem de Declaração

Se as regras tiverem a mesma especificidade e não houver !important, a última regra declarada no CSS terá prioridade.

Exemplo:

p {
  color: blue;
}

p {
  color: green;
}

Resultado: O texto será verde porque a última declaração de p tem prioridade.

Hierarquia de Prioridade (Resumo)

  1. Estilos inline (especificidade máxima).
  2. Regras com !important.
  3. Seletores de ID.
  4. Seletores de classe.
  5. Seletores de atributo, pseudo-classes e pseudo-elementos.
  6. Seletores de elemento (tag).
  7. Ordem de declaração (última regra aplicada).

Exemplo Completo

Vamos ver um exemplo que reúne todos esses conceitos:

<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    p {
      color: blue; /* Especificidade baixa */
    }

    .important-text {
      color: green; /* Especificidade média */
    }

    #unique-text {
      color: red; /* Especificidade alta */
    }

    p {
      color: purple; /* Ordem de declaração */
    }

    .important-text {
      color: yellow !important; /* Importância */
    }
  </style>
</head>
<body>
  <p id="unique-text" class="important-text" style="color: orange;">
    Texto de exemplo
  </p>
</body>
</html>

Resultado: A cor final do texto será amarela, porque a classe .important-text utiliza !important, que tem prioridade máxima, superando todos os outros estilos, incluindo o estilo inline.

3. Criando um botão com classes

.button {
  padding: 8px 32px;
  font-size: 18px;
  border-radius: 6px;
  border: solid 1px #000000;
  cursor: pointer;
}

.button:hover {
  opacity: 0.5;
}

.button:focus {
  outline: 3px solid green;
}

.button--default {
  background-color: #ddd;
  color: #333;
  border: solid 1px #000000;
}

.button--success {
  background-color: #e8ffdb;
  color: #1f9108;
  border: solid 1px #075a11;
}

.button--error {
  background-color: #ffcccc;
  color: #9e1010;
  border: solid 1px #5a0707;
}

.button--small {
  padding: 6px 18px;
  font-size: 12px;
}

4. Exemplos de tipos de display

1. Display: block

Elementos com display: block ocupam a largura total do contêiner pai e começam em uma nova linha.

.block {
      display: block;
      width: 100%;
      background: #add8e6;
      padding: 10px;
      margin: 5px 0;
  }
Block Element 1
Block Element 2

Exemplos de tags nativamente block:

  • div
  • h1 a h6
  • p

2. Display: inline

Elementos com display: inline ocupam apenas o espaço necessário e não começam em uma nova linha.

.inline {
  display: inline;
  background: #90ee90;
  padding: 10px;
  margin: 5px 0;
  }
Inline Element 1 Inline Element 2

Exemplos de tags nativamente inline:

  • span
  • a
  • img

3. Display: inline-block

Elementos com display: inline-block são como elementos inline, mas podem ter largura e altura definidas.

.inline-block {
  display: inline-block;
  background: #ffcccb;
  padding: 10px;
  margin: 5px 0;
  }
Inline-block Element 1
Inline-block Element 2

Nenhum elemento nativamente é inline-block, mas é possível alterar o comportamento de elementos block e inline para inline-block.

5. Conceitos e exemplos de position no CSS

A propriedade position no CSS é usada para especificar o método de posicionamento de um elemento. Existem cinco valores principais para position: static, relative, absolute, fixed e sticky. Cada um desses valores define um comportamento diferente para o posicionamento dos elementos na página.

1. Position: static

Este é o valor padrão. Elementos com position: static são posicionados de acordo com o fluxo normal do documento, sem nenhuma modificação adicional.

.static {
    background: #add8e6;
    padding: 10px;
    margin: 5px;
}
Static Element

2. Position: relative

Elementos com position: relative são posicionados de acordo com o fluxo normal do documento, mas podem ser deslocados do seu lugar original usando as propriedades top, right, bottom e left.

.relative {
    position: relative;
    top: 20px;
    left: 20px;
    background: #90ee90;
    padding: 10px;
    margin: 5px;
}
Relative Element

3. Position: absolute

Elementos com position: absolute são removidos do fluxo normal do documento e são posicionados em relação ao seu contêiner pai mais próximo que tenha uma posição diferente de static. Se não houver tal contêiner pai, o elemento será posicionado em relação ao <html>.

.container-relative {
    position: relative;
    background: #f0f0f0;
    padding: 20px;
}

.absolute {
    position: absolute;
    top: 10px;
    left: 10px;
    background: #ffcccb;
    padding: 10px;
}
Absolute Element

4. Position: fixed

Elementos com position: fixed são removidos do fluxo normal do documento e são posicionados em relação à viewport (a janela de visualização do navegador). Eles permanecem fixos em relação à viewport, mesmo quando a página é rolada.

.fixed {
    position: fixed;
    top: 0;
    right: 0;
    background: #add8e6;
    padding: 10px;
}

5. Position: sticky

Elementos com position: sticky são posicionados de acordo com o fluxo normal do documento até que um determinado ponto de rolagem seja atingido. Nesse ponto, eles "grudam" na posição especificada (top, right, bottom, left) e permanecem fixos até que seu contêiner pai esteja fora de visão.

.sticky {
    position: sticky;
    top: 0;
    background: #90ee90;
    padding: 10px;
    margin: 5px;
}
